passo 2 de 6

Instalar o
framework

Cole o comando abaixo no terminal do seu servidor. O instalador faz todo o resto automaticamente.

💡
Se estiver em uma VPS, primeiro conecte: ssh ubuntu@IP_DO_SERVIDOR
terminal 
# Cole no terminal e pressione Enter
curl -fsSL https://install.nucleoempreende.com.br | bash -s SUA_CHAVE_AQUI
🔑
Substitua SUA_CHAVE_AQUI pela chave de licença recebida no e-mail de compra.
Formato: NF-XXXX-XXXX-XXXX-XXXX
📋 O que o instalador faz
✓Valida sua licença
✓Instala Python 3.11+
✓Instala 25+ dependências
✓Configura Redis local
✓Cria o comando nucleo
✓Cria serviço systemd
✓Instala Playwright/Chrome
✓Abre o Setup Wizard
⏱ Tempo total: 5–8 minutos 📦 Tamanho: ~180 MB

passo 4 de 6

Testar as
conexões

Antes de iniciar, vamos confirmar que tudo está funcionando. Rode no terminal do servidor:

terminal
cd ~/Nucleo-empreende
python3 testar_tudo.py
testar_tudo.py — saída simulada
Clique em "Simular" para ver a saída esperada →
🔍 Como interpretar
✅ produçãoAPI key configurada e conexão OK
⚠️ simulaçãoSem key — funciona mas não faz chamadas reais
❌ erroKey configurada mas inválida — verifique o formato
⚠️
Para continuar você precisa de Gemini, Groq e Twilio em produção. Os demais podem ficar em simulação.
